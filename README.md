# 🔐 File Encryption Tool

A Python-based tool for encrypting and decrypting files using symmetric key cryptography.

---

## 📌 Project Overview

This project allows users to securely encrypt and decrypt files using a generated secret key. It demonstrates practical cryptography implementation and secure file handling techniques.

---

## 🚀 Features

- Encrypt files securely
- Decrypt files with the same key
- Uses symmetric encryption (Fernet / AES-based)
- Key generation support
- Lightweight and easy to use

---

## 🛠 Technologies Used

- Python 3
- Cryptography Library
- File Handling
- Symmetric Encryption

---

## 📂 Project Structure

file-encryption-tool/
│
├── encrypt.py
├── decrypt.py
├── key.key (ignored)
├── requirements.txt
└── README.md

---

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run encryption:
   python encrypt.py

3. Run decryption:
   python decrypt.py
